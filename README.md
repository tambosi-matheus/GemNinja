# GemNinja
 <sub> Totally not a Fruit Ninja ripoff <sub>

A slashing game focused on getting the biggest combo
[Download][APK]

### Programming Features
- Wave system
- Pools (gems, bombs, touch trails)
- Coroutines
- Random generation (position, force, rotation)
